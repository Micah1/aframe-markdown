# aframe-markdown

Renders Markdown using SDF text in a-frame

## Supported Markdown Features

- [x] Paragraphs
  - [ ] * Line breaks
  - [ ] Inline emphasis formatting
- [x] Headers
- [ ] Blockquotes
  - [ ] Blockquote styling
- [x] Images
- [x] Lists
  - [x] Nested lists
  - [x] Unordered lists
  - [ ] Ordered lists
- [ ] Horizontal rules
- [ ] Tables
- [ ] Code
  - [ ] Inline code
  - [ ] Code blocks
- [ ] Links
